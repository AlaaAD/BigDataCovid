BIG DATA PROJECT

Il s'agit d'un exemple très simplifié de l’utilisation des big data.

Nous avons choisi de prendre comme exemple des enregistrements de COVID-19 dans différents pays dans une période précis. Tous d’abord nous allons collecter une très petite partie des données de ce fichier Excel qui contient les données de chaque pays, nous allons ensuite les traiter et enfin les analyser.
